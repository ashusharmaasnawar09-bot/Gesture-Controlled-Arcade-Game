## OpenCV Introduction

OpenCV is an open-source computer vision library used for image processing and video analysis.

### Topics Learned

#### Image Processing

* Reading Images
* Displaying Images
* Resizing Images
* Cropping Images
* Rotating Images

#### Drawing Operations

* Drawing Lines
* Drawing Rectangles
* Drawing Circles
* Adding Text to Images

#### Video Processing

* Webcam Access
* Capturing Video Frames
* Displaying Live Webcam Feed
* Keyboard Event Handling using cv2.waitKey()

---

## Key Functions Learned

### Image Handling

* cv2.imread()
* cv2.imshow()
* cv2.imwrite()

### Drawing

* cv2.line()
* cv2.rectangle()
* cv2.circle()
* cv2.putText()

### Video

* cv2.VideoCapture()
* cap.read()
* cv2.waitKey()
